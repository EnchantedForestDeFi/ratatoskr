# Brand Asset Provenance

This file tracks where each AI-generated brand asset came from. It
exists for two reasons:

1. **Reproducibility.** If we ever need to regenerate at a higher
   resolution or produce a complementary mark, we need the exact
   prompt + product.
2. **IP defense.** US copyright currently does not protect AI
   outputs absent substantial human creative input, but Google's
   terms-of-service grant the prompt-author commercial usage rights.
   If anyone ever challenges our right to use these images, this
   file is the documentation trail.

---

## Format

For each committed asset, record:

- **Filename** (in `doc/brand/source/`)
- **Generated by** (which Google AI product, with URL or app name)
- **Account** (which Google account did the generation — for
  retrieval of the original session if needed)
- **Date** (ISO 8601)
- **Prompt** (the exact prompt text, copy-pasteable for
  regeneration)
- **Negative prompt or post-edits** (if any — e.g. "manually
  cropped 30px from bottom-right to remove watermark")
- **Notes** (anything else useful — alternative outputs, why this
  one was selected, related variants generated in the same session)
