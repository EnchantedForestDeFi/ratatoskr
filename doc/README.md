Ratatoskr Core
==========

This is the official reference wallet for Ratatoskr digital currency and comprises the backbone of the Ratatoskr peer-to-peer network. You can [download Ratatoskr Core](https://www.ratatoskr.org/downloads/) or [build it yourself](#building) using the guides below.

Running
---------------------
The following are some helpful notes on how to run Ratatoskr Core on your native platform.

### Unix

Unpack the files into a directory and run:

- `bin/ratatoskr-qt` (GUI) or
- `bin/ratatoskrd` (headless)

### Windows

Unpack the files into a directory, and then run ratatoskr-qt.exe.

### macOS

Drag Ratatoskr Core to your applications folder, and then run Ratatoskr Core.
